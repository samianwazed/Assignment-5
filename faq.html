Question-1: What are the different ways to select an element in the DOM?

There are Three ways to select an element in the DOM they're getElementById() , getElementsByTagName() , querySelector()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML sets HTML content, including tags. It allows inserting HTML. innerText sets only visible text, ignoring hidden elements and removing HTML tags. textContent sets all text, including hidden elements, and treats HTML tags as plain text

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for multiple child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept where an event that is triggered on an inner element propagates upward through its parent elements, all the way to the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, I will use document.createElement(); then, to add it to the DOM, will use appendChild(). To remove an element, will use removeChild() or remove().
